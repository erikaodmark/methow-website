Introduction

Thank you for purchasing Educomp.
It is a powerful theme and we are sure you will have an easy time customizing it according to your needs.

General

File Structure

EduComp's file structure is really simple.

		EduComp
		|- css/
                    |- Blue/
                    |- Green/
                    |- Red/
                    |- Yellow/
		|- fonts/
		|- images/
                    |- gallery/
		|- js/
		|- slider-revolution/
               *.*

EduComp folder contains all the files and folders used in this template. All the HTML pages directly reside within the EduComp folder.

js folder contains all the javascript files including the plugins (except the revolution slider files).

css folder contains all the css files including the plugins (except the revolution slider files).

images folder contains all the images used in EduComp. It also includes a folder "gallery" which contains folders based on "Albums" on Gallery Albums page.

fonts folder contains all the font files for icons. EduComp uses two icon font families namely Glyphicons and FontAwesome.

slider-revolution folder contains all the files used by revolution slider (js, css etc.)

Installation

Whether you are working on your local system or you need to upload the website to the server, you need to copy all the files and folders present in the EduComp folder

Use any FTP client to copy files from your system to the server. All the below folders and files need to be uploaded:

  1. css Folder
  2. fonts folder
  3. images Folder
  4. js folder
  5. slider-revolution folder
  6. All HTML files
Note : index.html is the homepage of your website which opens by default when someone visits your site.

HTML

HTML Structure

EduComp uses the following structure on almost all pages with exceptions like Coming Soon page.

                    
<html>
     <head>
          <title>EduComp Education Website Template</title>
          <!-- Here all the Stylesheets and meta tags are included along with the Title   -->
     </head>

     <body>
          <header>
               <!-- Topbar and Navigation come here -->
          </header>
          
          <div>
               ...
          </div>
                <!-- Content area -->
          <div>
               ...
          </div>
          
          <footer>
               <!-- Footer Items Come here -->
          </footer>
     </body>
</html>

head houses the Title, Meta tags, Includes like CSS stylesheets etc.

body houses the Header (Website Menu), Content of the page and the Footer (Blog lists, widgets etc.)

Page Header

Do Not confuse Header with Page Header Figure : Page Header

Page header contains the Heading which tells us which page we are on. Page Header is not present on Home page.

In EduComp it contains the following:

  1. Page Title/Page Name - Tells us which page we are currently on. Example : About Us
  2. Breadcrumb - Tells us our exact location within the directory structure of the website. Example : Home / Library / Data

Sidebar

Love sidebars? Use them on the website pages or on the blog pages, as you like.

The basic structure of sidebars in EduComp is as follows : 

<aside class="widget widget-specific-class">        
     <!-- Widget Content -->
</aside>

widget class is the wrapper class for every widget in the sidebar.

widget-specific-class class is widget specific class like twitter-widget, search-widget, category-widget etc. depending upon which widget you use.

Footer

Footer in EduComp is a full width footer spanning the whole width of the user screen.

Footer is EduComp is divided into 2 parts:

  1. Primary Footer - The upper big part of the footer. Houses the Footer widgets
  2. Secondary Footer - The lower smaller part of the footer. Houses Copyright info and Social Media Connect

The basic Footer structure in EduComp is as follows:

<footer class="footer">        

     <div class="footer-pri">
          <!-- Primary Footer - Footer Widgets Come Here -->
     </div>

     <div class="footer-sec">
          <!-- Secondary Footer - Copyright Info / Social Media -->
     </div>
     
</footer>

footer-widget class is the wrapper class for every widget in the sidebar.


Footer Widgets Housed in Primary Footer

The basic structure of Footer widgets in EduComp is as follows : 

<div class="footer-widget">        
     <!-- Widget Content -->
</div>

Customization

Changing Color Theme

EduComp comes with 4 color schemes built in:

  • Yellow (The default color theme)
  • Blue
  • Red
  • Green

If you want to change the color scheme of template, follow the given steps. As an example, we will be changing the color scheme to Green, but the steps are same for all other color schemes.

  1. In the template's Root directory, find the CSS folder and open it.
  2. The CSS folder contains some CSS files and Folders. The default color scheme of this template is Yellow.
  3. To change the color scheme to Green, open the folder named Green.
  4. Copy all the files in that folder.
  5. Paste the files in the CSS folder i.e one level above. Overwrite the files if warned by the system.

That's it!

JS

We have taken utmost care so that you never need to deal with the javascript files until and unless you need to customize the functionality. Thats why we have used data-attributes to simplify things as much as possible.

Overview of Main.js (js/main.js)

main.js is the custom javascript fil fo EduComp which contains all the initializing scripts etc.

To make handling simple, this file is divided into 2 parts:

  1. Function Calls
  2. Function Definitions

If you want to fiddle with certain values like timing or want custom functionalities, you may edit this file. The file is well commented.

Countdown Timer

Needs following files:
  • jquery.min.js
  • jquery.countdown.min.js
  • main.js

Educomp offers multiple instances of Countdown Timer.

We understand that dealing with Javascript can be messy. You can directly change the date for Countdown Timer with data attributes

The basic structure for timer is as follows:

<div class="timer" data-date="2016/06/20 06:34:23">
     <span class="days">0</span>days
     <span class="hours">0</span>hrs
     <span class="minutes">0</span>min
     <span class="seconds">0</span>sec
</div>

Simply edit the attribute data-date to your choice of date. Format for the date is YYYY/MM/DD hh:mm:ss. Thats it!

Google Map Custom contact.html

Needs following files:
  • jquery.min.js
  • Google Maps API
  • main.js

The map in EduComp is theme styled and has Auto Center feature.

The basic structure of Google map is as follows:

<div class="gmap" id="gmap" 
    data-lat="51.508103" 
    data-long="-0.074911" 
    data-info-win="<h6>Educomp London Campus</h6>">
</div>

To load the place of your choice, you need to do the following:

  1. Replace the value of data-lat attribute with your place's Latitude.
  2. Replace the value of data-long attribute with your place's Longitude.
  3. Replace the value of data-info-win attribute with the data you want to display in the info window that shows up on the marker. HTML is supported

How to find the Latitude and Longitude of your place?

  1. Open Google Maps
  2. Find your place on the map
  3. Right click on your place and Click "Whats here?"
  4. The Latitude and Longitude of the place will be shown.
  5. Put those values here.

Modal Pop Up (index.html)

Needs following files:
  • bootstrap.min.js
  • jquery.min.js
  • main.js
Figure : Youtube Video Theme Styled

EduComp offers a superb functionality of PopUps using Bootstrap Modals.

Any Modal can be used to make it "Pop Up" on Home Page or any other page of your choice.

You just need to use the following 2 attributes:

  1. data-open-onload Valid Values : "true" | "false"

    If you want to open your modal on page load then set the value as "true" else set it to "false".

  2. data-open-delay Specify time in milliseconds

    If you want that the Pop Up should come after some delay on page load, then specify time here in milliseconds

Modals can contain any HTMl data. So, you can modify according to your needs.
